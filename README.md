# Barnyard

https://barnyardproject.herokuapp.com/

## Description
An app that for those with farm animals to connect to others in the community.

## Built Using
* GraphQL
* Express.js
* Node.js
* MongoDB
* MERN
* Heroku

## Installation Instructions
1. run: git pull origin main in terminal
3. run: npm install
4. run: npm start
